# File Metadata Uploader

A full stack web application to upload files and display file metadata.

---

## Technologies Used

### Frontend
- React JS
- Tailwind CSS
- Axios

### Backend
- Spring Boot
- Java
- REST API

---

## Features
- Upload files
- Display file name
- Display file size
- Display file type
- Display uploaded time
- Responsive UI
